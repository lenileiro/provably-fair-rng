Provably Fair Verifier

Independently verify any game round outcome using HMAC-SHA256 — no login required.

How It Works

Algorithm: HMAC-SHA256
outcome = HMAC_SHA256(key=serverSeed, data="clientSeed:nonce")

Flow:
1. Before each round, the server commits SHA256(serverSeed) to the client.
2. The client provides their own clientSeed.
3. After the round, the server reveals serverSeed.
4. Anyone can verify: compute HMAC and compare to the recorded outcome.
